# EduHub - Online Course Management System

## Project Description
EduHub is a professional backend API for an online course platform built with **Spring Boot 3**.
The goal of this project is to apply modern Java development standards, including Clean Code, Layered Architecture, and RESTful API design.

### Key Features:
* **Course Management:** CRUD operations for online courses (In Progress).
* **Student Management:** Registration and management of participants (Planned).
* **Enrollment System:** Handling Many-to-Many relationships between students and courses (Planned).
* **Security:** Securing the API using Spring Security & JWT (Planned).

## Tech Stack
* **Java 17/21**
* **Spring Boot 3** (Web, Data JPA, Validation)
* **Database:** H2 (In-memory for development)
* **Build Tool:** Maven
* **Lombok**
